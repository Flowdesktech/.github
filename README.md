# .github

This repository powers the **Flowdesk** organization profile on GitHub.

- [`profile/README.md`](./profile/README.md) is rendered on [github.com/Flowdesktech](https://github.com/Flowdesktech) as the org landing page.

## How it works

GitHub automatically looks for a repository named `.github` inside an organization and, if it finds a file at `profile/README.md`, uses it as the public profile README shown at the top of the org page.

## Editing

1. Update [`profile/README.md`](./profile/README.md).
2. Commit and push to `main`.
3. Changes appear on [github.com/Flowdesktech](https://github.com/Flowdesktech) immediately.
